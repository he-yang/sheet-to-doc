# Sheet to Doc - English Documentation

This is the English documentation for Sheet to Doc, a tool that converts Excel sheets to professional documents.

## Documentation Structure

The documentation is built using Sphinx and Read the Docs. The main content is in the `source` directory, with Markdown files that cover various aspects of using Sheet to Doc.

## Building the Documentation

To build the documentation locally, you need to have Python and Sphinx installed. Then run:

```bash
pip install -r source/requirements.txt
sphinx-build -b html source build/html
```

The built documentation will be in the `build/html` directory.
