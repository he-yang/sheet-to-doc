# Data Security

Your data security is our top priority.

Our software does not collect your Excel, JSON, JSONL, CSV data, nor does it collect your template files. Our privacy policy details how we handle your data, see [Privacy Policy](privacy.md).

If you still have concerns about data security, we recommend that you use the offline desktop version, which can run on an intranet or on a standalone device without network connectivity.
